User Manual

Homepage

  • The homepage of Romeo online simulator consists of various components. There are two buttons sign in and register on the top right corner. Every new user needs to register a new account then sign in to start using our system.

  • Additionally, there are two buttons join today and contact us at the bottom of the homepage. The join today button is prompt user to register and sign in to start using our online simulator. The contact us button is offering user the chance to contact us privately for any suggestions or help.

Register

  • When user clicks register, it will show the user such interface. Users need to enter the username, email address and password to create a new account. Once the user enters all required information, he can click register button to finish it.

Sign in

  • Similar with the register interface. After user registers successfully, he can click the sign in button to jump to a new page which will ask user to enter such information. After the user enters all information, he can click the log in button to finish it.

Navigation bar

  • On the top left corner and the bottom of the home page, there are four options for user which are Home, Live Competition, Past Recordings, and Support Teams.

  • By clicking the home button, the user will jump to the home page. By clicking the Live Competition button, the user will be redirected to the competition page. If there is no ongoing competition, the user can choose the number of participants and the specific participant teams to start a new competition. Otherwise, if there is an ongoing live competition, the user will be able to watch the current ongoing match in the competition.

  • By clicking the Past Recordings button, the user can watch the past competition’s matches and download the recording.

  • By clicking the Support Teams button, the user can find the information like the background of each team and download the relative resources from each team.

Live Competition

  • When user clicks Live Competition button, if there is no ongoing competition, the webpage will show the above interface (Figure 14). Firstly, it will ask user to enter the number of participants and choose any teams that they prefer.

  • In addition, the user can choose random teams by pressing the dice icon next to “Participants:” label. After that, the user can click Start the competition button. If there are any errors related to the number of participants chosen, that error will be displayed above the button.

  • And the webpage will be reloaded for live competition stream. However, due to time constraint, the code is not fully implemented, therefore, currently the user will have to manually refresh the page.

  • Figure III shows the live competition wait for match. When the users wait for the match, they can send message and interact with other users in the public live chat by entering text in the text field and click send button to send the message.

  • Due to time constraint for code implementation, the user will have to refresh the page manually to load the match when it’s done. It will be fixed in the next update.

  • Right below the live competition stream, there is a leader board that shows the result of each match as well as the bracket as the competition progress.

  • Figure 18 is the current match running after the match loads successfully. There are 11 players in each team. The two-team player is presented by different colour (yellow and blue).

  • In addition, the goalkeeper has different colour in order to help user to easier identify them.

Past Recordings

  • When user clicks past recordings button, the system will show the above interface (figure 19). Each competition contains two rounds or three rounds depends on the number of participants in each competition.

  • The user can play the recording of each round by clicking the arrow icon or click the download icon to download the log file of any competition they are interested in.

  • Figure 20 is the interface when user clicks the play recording icon. The same virtual court as the live competition will be called.

Support Teams

  • Figure 21 shows up when user clicks the support teams’ button. The webpage shows the user all teams’ details like the background and the logo of each team.

  • At the bottom of each team description, there is a link that user can click to download the related resources for each team.

  • There are 12 teams available but their resources are not available to download in this release yet.
